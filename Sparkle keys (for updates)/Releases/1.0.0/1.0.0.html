New in CocosBuilder 1.0

Important! CocosBuilder 1.0 has a new license. The program is still free to download, but if you use it to create apps that you upload to AppStore, a donation of $20 or more is required. If this is not acceptable to you, please do not upgrade!

Lots of small and big bug fixes.

CocosBuilder now uses much less CPU.

Don't forget to download the new example files and improved CCBReader from www.cocosbuilder.com!
