New in CocosBuilder 0.4

CocosBuilder now has support for multiple documents. Note that there are still no indications that a document has been edited, so remember to save them. (This will be fixed in CocosBuilder 0.6)

Support for sprite sheets and jpg images has been added

Lots of stability fixes, and the look of the application has got a face lift!

From now on, new versions of CocosBuilder will be able to read files from version 0.4 and newer.
